Описание:
----------------------
Программа предназначенная для преобразования фотографии в ASCIIart или ANSIart
----------------------

Требования
•Python версия не ниже 3.6
•модуль termcolor версия 1.1.0
•модуль Pillow версия 6.2.1

Использование:
 Через консоль:
      python Start_Program.py --image=image, --weidth=new_weidth, --type = type

type:
1 - ASCIIart вывод текстовым файлом
2 - ANSIart вывод в консоль
3 - ASCIIart вывод png файлом
